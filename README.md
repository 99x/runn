# runn

Make your own terminal aliases easily!

## Install it

```
$ npm i -g runnit
```

## Save it

```
$ runn cu "curl https://raw.githubusercontent.com/shalithasuranga/runn/master/bin/runn.js"
```

## Run it!

```
$ runn cu
```
