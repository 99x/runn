## Getting Started

### Prerequisites

You need to install node package manager to runn this!

* npm

```sh
npm install npm@latest -g
```

### Installation

1. Install the NPM packages

```sh
$ npm i -g runnit
```

2. Save it

```sh
$ runn cu "curl https://raw.githubusercontent.com/shalithasuranga/runn/master/bin/runn.js"
```

## Usage

### runn it!

```sh
$ runn cu
```
